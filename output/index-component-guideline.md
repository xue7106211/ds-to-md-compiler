## 二、组件构成

| 序号 | 主要构成 | 区域概述 |
| --- | --- | --- |

## 三、组件分类及交互规则

### 全显索引

- 悬浮在内容之上，默认选择"全显索引"。
- 当容器高度不够展示时，"全显索引"转换为"半显索引"。
- 高度可根据页面高度调整，压缩高度后，需对应减少内容透出，省略部分使用 `·` 代替。

**交互规则**

- 垂直滑动选择锚点，或直接点击锚点，页面滚动到对应位置。
- 按下或滑动时，**放大展示**锚点内容。
- 贴于页面右侧，RTL 布局中相反。
- 顶部为开头锚点，底部为结束锚点。
- 索引在内容区**上下居中对齐**。

### 半显索引

- 当容器高度不够展示全部索引项时自动切换。
- 省略部分使用 `·` 代替。

![全显索引示意：所有锚点完整展示，悬浮在内容右侧](图像占位符)

![半显索引示意：部分锚点被省略，以点号代替](图像占位符)

```yaml
interaction_rules:
  variants:
    - id: full_display
      label: 全显索引
      default: true
      condition: 容器高度足够展示所有锚点
      position: floating_above_content
    - id: partial_display
      label: 半显索引
      condition: 容器高度不够展示时自动切换
      omitted_items_placeholder: "·"
  selection:
    vertical_swipe: true
    tap_anchor: true
    page_behavior: scroll_to_position
  feedback:
    on_press_or_swipe: enlarge_anchor_content
  layout:
    horizontal: right_aligned
    rtl: left_aligned
    vertical: center_in_content_area
    top_anchor: first_item
    bottom_anchor: last_item
```

## 待确认问题

1. **问题位置**：文档元信息 — 更新记录  
   **问题描述**：更新记录第 2 条写为"标题字数限制变更为 `XX` 字"，其中 `XX` 为占位符，未填写实际数值。  
   **需要你确认**：标题字数限制的实际值。

2. **问题位置**：第四节「1. 主题」— 深色模式字色  
   **问题描述**：深色模式字色标注为 `#000000 50%`（黑色 50% 透明度），在深色背景 `#191919` 上几乎不可见。通常深色模式文字应为白色系。  
   **可能解释**：原文可能为 `#FFFFFF 50%`，文本提取时 `#` 后颜色值丢失了 `FFFFFF` 前缀，或为 PDF OCR 错误。  
   **需要你确认**：深色模式字色的实际色值。

3. **问题位置**：第四节「3. 样式」— 操作规格  
   **问题描述**：样式表中「操作规格」行的内容完全缺失，无法提取索引项的字号、字重、行高、间距等参数。  
   **需要你确认**：请提供第 5-6 页截图或补充索引项的排版参数（字号、字重、单项高度、间距等）。

4. **问题位置**：第三节 — 半显索引  
   **问题描述**：半显索引的交互规则表内容几乎完全缺失，仅提取到标题。全显→半显的切换阈值（容器高度临界点）也未定义。  
   **需要你确认**：半显索引是否有独立的交互规则，以及全显/半显的切换阈值。

5. **问题位置**：第四节「4. 热区」  
   **问题描述**：热区宽度仅 `21dp`，远小于通常推荐的最小触摸目标。这是否为整个索引条的可触摸区域宽度，实际触摸区域是否有扩展。  
   **需要你确认**：`21dp` 是视觉宽度还是触摸目标宽度，实际可触摸区域是否更大。
