## 二、组件分类及交互规则

以下将 PDF 中的复合表拍平为二维表，便于检索与代码生成。

### 2.1 终端与类型总览

| 终端 | 类型 | 整体样式规则 | 各区域交互说明 | 最佳实践 | 补充描述 |
| --- | --- | --- | --- | --- | --- |
| 手机/折叠/Pad | 单组选择器 | 见下表「单组-样式规则」 | 见下表「单组-使用规则」 | 线上最佳实践案例 | — |
| 手机/折叠/Pad | 多组选择器 | 见下表「多组-样式规则」 | 见下表「多组-使用规则」 | 线上不符合的案例（可选） | — |

### 2.2 单组选择器

| 维度 | 内容 |
| --- | --- |
| **使用场景** | 只有一组选项，如目标温度调节。 |
| **样式规则** | 当选项为 1 列时，单位居于选项右侧。 |
| **使用规则** | 1. 垂直滚动选项列，支持加速度滚动和单格滚动，可配合音效与振动。<br>2. 每列选项展示数量为奇数，位于垂直中心的选项为选中选项。<br>3. 单位不支持滚动。<br>4. 可用于页面，也可嵌入其他容器（如弹窗、浮窗），需配合提交按钮使用，提交后所选项生效。 |

### 2.3 多组选择器

| 维度 | 内容 |
| --- | --- |
| **使用场景** | 用于多组选项选择，如日期选择。 |
| **样式规则** | 当选项为 2 列以上时，单位居于顶部。 |

### 2.4 单位与列数规则（通用）

| 条件 | 规则 |
| --- | --- |
| 选项为 1 列时 | 可在选项后添加单位。 |
| 选项小于 3 时 | 优先不添加单位；如必须添加，在顶部加。 |
| 选项大于 3 时 | 不可添加单位；如不添加影响可用性，则拆分成多个选择器。 |

## 四、组件视觉规则

- **主题**：浅色主题、深色主题
- **布局**：水平对齐方式、垂直对齐方式
- **标题区**：字号、字重、字数、字色、对齐方式、带跳转的点击区域热区
- **辅助描述区**：字号、字重、字色、对齐方式、带跳转的点击区域热区
- **操作区**：图标尺寸、间距、边距、点击热区
- **动效**：动效样式

### 2. 主题

| 分类 | 浅色模式 | 深色模式 |
| --- | --- | --- |
| 主题（常规态）描述 | 选中数据颜色：`#3482FF` 100% | 选中数据颜色：`#277AF7` 100% |
| 未选中数据 | 未选中数据颜色：`#000000` 80% | 未选中数据颜色：`#FFFFFF` 80% |

```yaml
theme_tokens:
  light:
    selected: { hex: "#3482FF", alpha: "100%" }
    unselected: { hex: "#000000", alpha: "80%" }
  dark:
    selected: { hex: "#277AF7", alpha: "100%" }
    unselected: { hex: "#FFFFFF", alpha: "80%" }
```
